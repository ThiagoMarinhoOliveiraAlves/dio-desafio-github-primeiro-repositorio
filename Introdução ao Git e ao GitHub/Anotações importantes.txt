O Git Bash é um terminal extendido para otimizar o uso do Git.

Funcionalidade de alguns comandos essenciais no Git.

► git clone: Clona um clone do repositório que fica no GitHub(site), na máquina atráves do link do repositório(HTTPS/SHH).

► cd : Utilizado para entrar dentro de um repositório exemplo: "cd repositorio-git".

► git status .: Verifica os arquivos na branch.

► git add . or -a: Adiciona o arquivo ao controle de versão do Git(local).

► git commit -m: Utilizado para adicionar um comentário ao commit exemplo: git commit "Receita de Bolo".

► git push origin main .: Esse comando envia todos os commits para a nuvem no Git.

► git push -u origin main .: Esse comando envia todos os commits para a nuvem no Git e atualiza o arquivo modificado localmente.
